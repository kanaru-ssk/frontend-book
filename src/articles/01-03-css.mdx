## CSSを表示させる
CSSをwebページで表示するにはHTMLでCSSファイルを読み込む必要があります。

headタグの中にlinkタグを追加し、作成したCSSファイルのリンクを追加します。
```html
<!doctype html>
<html lang="en">
  <head>
    <meta charset="utf-8" />
    <meta name="viewport" content="width=device-width" />
    <link rel="stylesheet" href="./style.css" />
    <title>Hoge Coffee</title>
  </head>
```

#### ページ全体のスタイルを設定

bodyタグにスタイルを追加します。
```css
body {
  margin: 0;
  font-family: Helvetica, sans-serif;
  color: #cfc9c9;
  background-color: #272d31;
  text-align: center;
}
```
- `<margin>`: 余白を設定する。
- `<font-family>`: テキストのフォントを設定する。左から順に採用される。
- `<color>`: テキストの色を設定する。色名やrgbや16進数で指定する。
- `<background-color>`: 背景の色を設定する。色名やrgbや16進数で指定する。
- `<text-align>`: テキストを左や中央、右などに揃える。

#### 見出しのフォントサイズを設定

見出しであるh1、h2タグのテキストのフォントサイズを設定します。
```css
h1 {
  font-size: 48px;
}

h2 {
  font-size: 20px;
}
```
- `<font-size>`: フォントサイズを設定する。

#### メインコンテンツ全体のスタイルを設定

mainタグにスタイルを追加します。
```css
main {
  max-width: 1024px;
  margin: 0 auto;
}
```
- `<max-width>`: 横幅の最大値を設定する。

#### リンクの色を変更する

aタグのテキストの色を変更します。
```css
a {
  color: #85a3b7;
}
```

#### トップページを作成する
```css
.top-page {
  width: 100vw;
  height: 100vh;

  background-image: url(./top-page.png);
  background-size: cover;
  background-position: center;

  display: flex;
  justify-content: center;
  align-items: center;
}
```
- `<width>`: 横幅を指定する。
- `<height>`: 高さを指定する。
- `<background-image>`: 背景の画像を指定する。
- `<background-size>`: 背景の大きさを指定する。
- `<background-position>`: 背景の位置を指定する。
