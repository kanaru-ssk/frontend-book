# JavaScriptで動きをつける

## JavaScriptとは

JavaScriptは、webページ上に複雑なものを実装することを可能にするプログラミング言語です。

webページが表示されるたびに静的な情報が表示されるだけでなく、タイムリーにコンテンツを更新したり、インタラクティブマップや 2D/3D グラフィックのアニメーションを表示したりします。

## JavaScriptファイルを作ろう

VSCodeの左上のメニューから、File → Open Folderを選択し、前回の演習1で作成した`exercises-1`フォルダを開いでください。

VSCodeのファイル作成ボタンをクリックし、`script.js`ファイルを作成します。

<Image src="/img/01-04-js-02.png" alt="screenshot" width={512} height={479} />

## JavaScriptを書いてみよう

先ほど作成した`script.js`に、以下のコードを記述して下さい。

```js
alert("Alert From JavaScript!");
```

ブラウザを再読み込みするとアラートを表示するコードですが、何も起きません。

これは、`index.html`側で`script.js`を読み込んでいないためです。

以下のコードを`index.html`の`head`タグの中に追加しましょう。

```html
<script src="./script.js"></script>
```
