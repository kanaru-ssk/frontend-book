# HTMLでwebサイトの構造を作る

## 目標

画像のようなwebページを作成し、テキスト、リンク、リスト、画像など、基本的なHTMLの書き方を習得しましょう。

<Image src="/2024/img/01-02-html-01.png" alt="html logo" width={512} height={350} />

## HTMLとは

HTML（HyperText Markup Language）は、webページを見出しや段落、箇条書きなどに構造化するためのマークアップ言語です。

```html
<body>
  <h1>HTMLのイメージ</h1>
  <p>このようなコードを書く</p>
</body>
```

前提知識として、HTML要素は、コンテンツを開始タグと終了タグで囲む構造を持ちます。

<Image src="/2024/img/01-02-html-04.png" alt="screenshot" width={512} height={354} />

開始タグは山括弧でタグ名を囲った構造(`<タグ名>`)、終了タグはスラッシュ+タグ名を山括弧で囲った構造(`</タグ名>`)を持ちます。

要素には属性 (attribute) を設定し、要素に関する追加情報を含ませることができます。

<Image src="/2024/img/01-02-html-05.png" alt="screenshot" width={512} height={354} />

先ほどのコードの1行目を見てください。

{/* prettier-ignore-start */}
```html
<!DOCTYPE html>
```
{/* prettier-ignore-end */}

`<!DOCTYPE html>`は、この文書がHTML5という種類のHTMLドキュメントであることをブラウザに伝えるための宣言です。

```html
<html lang="ja"></html>
```

`html`タグは、HTMLドキュメント全体を囲むタグです。`lang`属性に`ja`が設定されており、このページが日本語で書かれていることを示しています。

```html
<head>
  <meta charset="utf-8" />
  <meta name="viewport" content="width=device-width" />
  <title>Title</title>
</head>
```

`head`タグの中には、ウェブページに表示しないコンテンツを記述します。

- `<meta charset="utf-8" />` : このページの文字エンコーディングがUTF-8であることを示します。これは、日本語などの多バイト文字を正しく表示するために必要です。
- `<meta name="viewport" content="width=device-width" />` : スマホなどで表示した時に適切なサイズで表示するための設定です。
- `<title>Title</title>` : このページのタイトルを定義します。これは、ブラウザのタブやブックマークなどに表示されます。

titleタグの中身を`<title>テスト</title>`のように書き換えて、保存した後にブラウザを再読み込みするとタブの文字が**テスト**に書き換わるか確認してみてください。

```html
<body>
  <h1>Hello World</h1>
</body>
```

`body`タグの中には、ウェブページに表示するコンテンツを記述します。

`h1`タグは、ページの中で1番大きい見出しを表示するためのタグです。

hは**heading**の略で、見出しという意味です。

`h1`〜`h6`まであり数字が大きいほど上位の見出しです。

```html
<!-- head : webページに表示しないコンテンツ -->
```

`<!--` と `-->`で囲まれた部分は、コメントと呼ばれ、ブラウザに表示されません。

主な役割は以下の3つです。

- 開発時のメモ: コードの意図や仕様変更履歴などを残すことができます。
- デバッグ: 問題のある部分を一時的に無効化し、原因を特定するのに役立ちます。
- コードの整理: 使用していないコードや将来使用する予定のコードを一時的に非表示にすることができます。

では、他のHTMLタグも書き足してみましょう。

### pタグ

  pタグで囲まれたテキストは、ブラウザ上では通常、1行分の改行が挿入され、段落として表示されます。

### imgタグ

{/* prettier-ignore-start */}
<img src="/sample/cup.png" alt="cup" className="m-0 w-56" />
<a href="/sample/cup.png" download>cup.png</a>
{/* prettier-ignore-end */}

上のリンクをクリックし、`cup.png`をダウンロードしてください。

ダウンロードフォルダの`cup.png`を、`web`フォルダに移動します。

`index.html`に以下のように記述しましょう。

```html
<img src="./cup.png" alt="カップ" width="480" />
```

imgは**image**の略で、webページに画像を埋め込みます。

`src`属性に、埋め込みたい画像へのパスを指定します。

`alt`属性に、画像のテキストによる説明を指定します。
`alt`属性は必須ではありませんが、画像がリンク切れした時に代わりに表示されたり、スクリーンリーダーで読み上げられたりします。

`width`属性に、画像の横幅を指定します。
`width`属性も必須ではありませんが、これを削除すると`cup.png`の元の大きさで表示されます。

### divタグ

今、ブラウザの表示が以下のように、画像の左下にリンクが来てしまっているかと思います。

もし縦に並んでいたら、ウィンドウサイズを広げると横に並びます。

<Image src="/2024/img/01-02-html-08.png" alt="screenshot" width={512} height={358} />

HTML要素には、**ブロックレ**ベルコンテンツと、**インライン**レベルコンテンツが存在します。

ブロックレベルコンテンツは、独立したブロックとして表示され、前後に改行が入ります。

インラインレベルコンテンツは、文章中などに挿入され、前後に改行が入りません。

これまで出てきた、

- `h1`
- `p`
- `ul`
- `ol`
- `li`

はデフォルトでブロックレベルコンテンツとしてレイアウトされますが、

- `a`
- `img`

はインラインレベルコンテンツとしてレイアウトされるため、横並びに表示されています。

デフォルトでブロックレベルコンテンツである`div`タグで`a`タグを囲むことで、リンクの前後で改行されます。

```html
<div>
  <a
    href="https://developer.mozilla.org/ja/docs/Learn/HTML/Introduction_to_HTML"
  >
    HTML 入門 - ウェブ開発を学ぶ | MDN
  </a>
</div>
<img src="./cup.png" alt="カップ" width="480" />
```
