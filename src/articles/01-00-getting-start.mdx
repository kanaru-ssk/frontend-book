## 目標

- Visual Studio Codeの基本的な使い方が分かる
- HTMLを使ってwebページの作成ができる
- CSSを使ってwebページの見た目を変更できる
- JavaScriptを使ってHTMLの内容を更新できる

## もくじ

<Link href="./getting-start/installation">**1 開発環境の構築**</Link>

まずはコードを書くためのテキストエディタをインストールします。

<Link href="./getting-start/html">**2 HTMLでwebサイトの構造を作る**</Link>

HTML（HyperText Markup Language）は、Webページの構造を定義するための言語です。ここでは、HTMLの基本的な要素や構造について学びます。

<Link href="./getting-start/css">**3 CSSで見た目を綺麗にする**</Link>

CSS（Cascading Style Sheets）は、Webページのスタイルやデザインを管理するための言語です。ここでは、CSSの基本的な構文、セレクタ、プロパティ、値などについて学びます。

<Link href="./getting-start/javascript">**4 JavaScriptで動きを付ける**</Link>

JavaScriptは、Webページの動的な振る舞いを実装するためのプログラミング言語です。ここでは、JavaScriptの基本的な構文、変数、データ型、制御構文などについて学びます。

<Link href="./getting-start/javascript">
  **5 GitHub Pagesでページを公開する**
</Link>

GitHub Pagesは、ウェブサイトを公開できるホスティングサービスです。ここでは、作ったwebページを実際にインターネット上で一般公開します。
